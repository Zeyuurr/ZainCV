Required accessories

Firewall SolarWinds
Text editor Aodbe Dreamweaver CC
Webserver Apache HTTP Server Project
This project is to create a web page for myself, which is convenient for me to display my personal information in interviews, making friends and making speeches. Website creation is divided into Analyze, Collect data, Design, Make a draft, Make Website five steps. After the completion of the website will include the website server, file server, database, firewall and other modules, the website is written in HTML and the use of external CSS to modify.
